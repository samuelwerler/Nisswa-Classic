# ⛳ Nisswa Classic - Golf Trip Manager

A simple, web-based golf trip management tool for organizing your group golf outings. Perfect for coordinating participants, tracking scores, and managing course schedules.

## Features

✅ **Trip Overview** - Set trip name, location, and dates
✅ **Participant Management** - Add up to 16+ golfers with confirmation status
✅ **Golf Courses** - Schedule multiple courses with dates and tee times
✅ **Leaderboard** - Track scores for multiple rounds and automatic ranking
✅ **Trip Notes** - Keep group notes and important updates
✅ **Data Persistence** - All data saved locally in your browser
✅ **Export** - Download your trip data as JSON
✅ **Fully Responsive** - Works on desktop, tablet, and mobile

## How to Use

1. **Open `index.html`** in any web browser
2. **Set Trip Details** - Enter trip name, location, and dates
3. **Add Participants** - Add each golfer and mark their status (Confirmed/Tentative/Declined)
4. **Schedule Courses** - Add golf courses with dates and tee times
5. **Track Scores** - Enter scores for each round in the leaderboard
6. **Add Notes** - Keep track of important information and updates
7. **Export** - Download your data at any time

## Data Storage

- All data is stored locally in your browser using localStorage
- **No account needed** - just open the file and start using it
- Data persists when you close and reopen the browser
- Use the export feature to backup your data
